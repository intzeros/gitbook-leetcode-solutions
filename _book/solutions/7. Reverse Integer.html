LeetCode 7. Reverse Integer

LeetCode 7. Reverse Integer

Reverse digits of an integer.

Example1: x = 123, return 321 Example2: x = -123, return -321


注意细节：

(1) 前置0

(2) 超出int范围（输出0）

class Solution {
public:
    int reverse(int x) {
        long long rst = 0;
        while(x){
            rst = rst*10 + x%10;
            x /= 10;
        }
        int MAX = 0x7fffffff, MIN = 0x80000000;
        if(rst > MAX || rst < MIN) return 0;
        return rst;
    }
};
